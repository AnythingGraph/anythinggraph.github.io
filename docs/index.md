# Anything Graph CLI

Build, query, and explore graph data from the command line.

Anything Graph CLI helps you connect data sources, model relationships, and run graph queries without leaving your terminal.

Anything Graph provides **policy as code**, **entity and relationship mapping**, and **ReBAC** (relationship-based access control) so AI agents can work with your data safely and with clear context.

## Project links

- [AnythingGraph website](https://www.anythinggraph.com/) — product overview and getting started
- [anything-cli on GitHub](https://github.com/AnythingGraph/AnythingGraph) — open-source CLI, reasoning service, and MCP server

